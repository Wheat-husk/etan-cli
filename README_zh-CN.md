# Etan-cli

使用其他语言阅读：[English](./README.md) | 简体中文

Etan CLI是一个命令行界面工具，可以帮助您初始化、开发和维护您的electron应用程序。它以多种方式提供帮助，包括搭建项目，在开发模式中提供服务，以及为生产分发构建应用程序。
